# TuZiFundTool
兔子基金小工具(Fund gadgets)，可搜索添加基金，查看基金实时数据，添加桌面小部件(You can search and add funds, view real-time data of funds, and add desktop widgets)

### 最新版本号：1.0.4(2020.05.22)

[软件下载地址](http://106.12.95.89/upload/TuZiFound.apk)
